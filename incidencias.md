# Incidencias

En este apartado se muestran las incidencias, es decir, las notas que se agregan en el calendario dentro del [perfil del alumno](lista-de-alumnos/perfil-del-alumno.md).

{% hint style="info" %}
Las notas no se pueden editar ni eliminar.
{% endhint %}

# Nuevo libro

Esta opción permitirá registrar un nuevo libro al catálogo para que los alumnos puedan registrar las lecturas realizadas de dicho libro. Al dar click se desplegará un formulario solicitando la siguiente información:

* Nombre del libro
* Autor
* Número de páginas
* Nivel del libro
* Número del libro

{% hint style="info" %}
Esta opción debe usarse **únicamente** en caso de que se añada un nuevo libro a la lista de lecturas recomendadas oficial de Kumon. Esto deberá ser notificado por la Orientadora antes de agregar un nuevo libro.
{% endhint %}

# Reporte mensual

{% hint style="warning" %}
Esta característica se encuentra actualmente en desarrollo y puede contener errores.
{% endhint %}

{% hint style="info" %}
Por el momento, el uso de esta opción está restringida a Soporte Técnico.
{% endhint %}

Esta opción generará un reporte mensual del mes seleccionado.

Al dar click, se desplegará un formulario solicitando la siguiente información:

* Mes: Un selector para indicar el mes.
* Año: Por defecto, el año en curso

<figure><img src="../../../.gitbook/assets/image (30).png" alt=""><figcaption><p>Interfaz del generador de reporte mensual</p></figcaption></figure>

Al dar click en el botón "Generar" se descargará automáticamente un archivo en formato `.csv` .
